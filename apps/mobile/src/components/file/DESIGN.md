# Git Management Interface Design System

- **Platform:** Mobile-first
- **Theme:** Light, clean, modern, spacious
- **Background:** Soft light grey (#f3f4f6)
- **Surface:** White (#ffffff) for file cards or rows
- **Primary Accent:** Green (#22c55e) for primary actions like "Commit" or text on secondary buttons
- **Text Primary:** Dark Grey/Near Black (#111827) for file names and headers
- **Text Secondary:** Medium Gray (#6b7280) for abstract indicators and sub-headers
- **Text Tertiary:** Light Gray (#9ca3af) for "Commit message..." placeholder and subtle accents
- **Status Badges:** Purple (#a855f7) or soft grey for "U" (Untracked) badges
- **Buttons (Primary):** Solid green background with contrasting white text/icon, rounded edges (8px)
- **Buttons (Secondary/Stage All):** Outlined or soft grey background (#f3f4f6) with green text
- **Layout:** Generous horizontal margins and 12px-16px vertical padding in rows
